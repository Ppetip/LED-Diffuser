How this connection works
  1. The GitHub Pages site is delivered over HTTPS.
  2. Your browser asks you to choose the nearby device named LED-Diffuser.
  3. The site connects to its BLE UART service.
  4. Pressing “Send” writes a compact JSON command in small BLE packets.
  5. The ESP32 renders the animation locally, so the page can disconnect afterward.

Web Bluetooth is not available in every browser. iPhone/iPad Safari does not provide this API; use Chrome/Edge on a computer or compatible Android browser. Bluetooth also requires this repository's firmware to be flashed first.
